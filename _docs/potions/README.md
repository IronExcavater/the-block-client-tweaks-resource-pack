# Modded Potions

This pack only adds custom textures for modded potion IDs. Vanilla potion IDs fall back to Minecraft/Xali resource-pack handling.

| Mod | Potion IDs | Base style |
| --- | --- | --- |
| Illager Invasion | `berserking`, `long_berserking`, `strong_berserking` | Xali Strength bottles with red slash motif |
| Rustic Delight | `haste`, `long_haste`, `strong_haste` | Xali Swiftness bottles with gold spark motif |
| Friends & Foes | `reaching`, `long_reaching`, `strong_reaching` | Xali Leaping bottles with green reach motif |
| Glow Up | `revealing`, `revealing_long`, `revealing_strong` | Xali Night Vision bottles with blue eye motif |

Generated paths:

- `assets/blockheads/textures/item/potions/*.png`
- `assets/blockheads/models/item/potions/*.json`
- `assets/minecraft/items/potion.json`
- `assets/minecraft/items/splash_potion.json`
- `assets/minecraft/items/lingering_potion.json`
